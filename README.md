# File transfer

Transfer multiple file type

## Installation

You should install [Maven](https://mkyong.com/maven/install-maven-on-mac-osx/) before run this web application .

Run client

```bash
cd client

npm install

npm start
```

Run server

```bash
cd server

mvn spring-boot: run
```

## Usage

App running on [localhost:3000](localhost:3000)
